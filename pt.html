Projetos

The Awaken web app allows the user to set the date/time they would like to wake up, and what information they would like to have at their disposal as they wake up, in the form of widgets (news, weather, commute time, etc). When set time comes, the display in which the Rasberry Pi is connected to turns on with a personalized display with the information. The display automatically turns off after a set amount of time.

  • Tech Stack: Ruby/Rails, HTML/CSS, Javascript/JQuery, Python, Rasberry Pi, Pub/Sub, Heroku, multiple APIs.

  • Developed in 04/2017

Orbidia connects you to the world via media. The web app continually suggests music and films based on the countries/language and genres the user chooses. Other users from around the world help you connect to the media that you want to consume by making comparisons between artists around the world (for eg, for those who like Artic Monkeys and are looking for brazilian music, how about Los Hermanos?).

  • Tech Stack: HTML/CSS, Javascript/JQuery, Ruby, Sinatra, Spotify API, IMDB API.

  • Developed in 03/2017

KeepYourCopy is a Chrome Extension that stores the last 15 bits of information the user has copied into his/hers computer's clipboard (Ctrl+C/⌘+C). In 'script' mode, everything the user copies to the clipboard goes to a Google Docs file automatically.





  • Tech Stack: HTML/CSS, Javascript, Google Drive API.

  • In production since 04/2017
